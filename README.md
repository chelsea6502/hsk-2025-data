# HSK 2025 Data

The latest 2025 HSK (Hanyu Shuiping Kaoshi) vocabulary, characters, and grammar data in TSV format, sourced from [chinesetest.cn](https://www.chinesetest.cn).

## Files

| File | Records | Description |
|------|---------|-------------|
| `vocabulary.tsv` | ~11,000 | Words with pinyin, part of speech, and level |
| `grammar.tsv` | ~593 | Grammar points with examples |
| `hanzi.tsv` | ~4,288 | Individual characters by level |

## Levels

HSK 3.0 has 7 levels: 1-6 plus 7-9 (combined). ~11,000 words total.
